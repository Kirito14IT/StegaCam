## 1. 行业痛点与解决方案 (Pain Point & Solution)

### 1.1 痛点：数字内容盗用与确权困难

随着图片在社交媒体、短视频平台与AI训练中的广泛使用，图像作品的版权盗用日益猖獗。创作者面临以下问题：
- 传统明水印易被裁剪、覆盖或修改
- 现有隐形水印在经过网络平台压缩、截图、二次编辑等常见处理后，**鲁棒性（抗干扰能力）严重不足**
- 无法提供简洁且可复现的 **有效溯源证据**
这些问题导致创作者难以取证和追溯，极大地损害了内容创作者和版权方的利益。

### 1.2 解决方案：隐溯盾

隐溯盾是一款**专为iOS/Android移动平台**设计的**新一代数字版权标识系统**。它通过AI算法将创作者的加密ID以**不可感知且高鲁棒性**的方式嵌入图像空间域，形成永久的“数字身份证”。

* **智能嵌入：** 用户通过App直接拍照或选取图片，即时将个人ID安全加密嵌入。
* **鉴权解码：** 随时对带水印图片进行快速身份验证，实现**图像版权追溯**。
* **强力保护：** 确保水印在经过主流社交媒体压缩、模糊、裁剪等干扰后仍能**精准解码**。

## 2. 核心技术与创新壁垒 (Core Technology & Innovation)

### 2.1 技术路线：深度学习隐写与跨平台部署

隐溯盾的核心技术是基于**自研的深度学习卷积神经网络（Convolutional Neural Network）架构**，驱动鲁棒型数字水印网络。该技术栈专注于解决鲁棒性与不可视性之间的复杂平衡。

### 2.2 核心创新与技术壁垒

| 创新点 | 描述 |
| :--- | :--- |
| **高强度鲁棒性** | 完成V1.0迭代，实现**JPEG有损压缩**，以及**90度增量旋转**干扰下，解码准确率仍维持在**98%以上**。此性能是项目核心壁垒。 |
| **移动端轻量化** | 采用**TensorFlow Lite**、等轻量化框架部署模型，结合跨平台技术**React Native**、**FastAPI**，实现App在移动设备上的**快速实时处理能力**。 |
| **未来确权机制** | 规划将**区块链ID**作为隐形水印的载体，实现水印信息写入后的不可篡改性和权威确权。 |
